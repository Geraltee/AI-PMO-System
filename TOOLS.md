# TOOLS.md - Local Notes
## Web Search Tools
When it comes to web search tools such as browser, you can only use the website accessible in Chinese Mainland.you should Prioritize using Baidu search engine.Some websites is prohibited like https://www.google.com  which is not accessible in Chinese Mainland.

## Conceptual differentiation
### Google Chrome vs Google Search Engine
Google Chrome is not equivalent to Google Search Engine, opening Google Chrome does not mean opening the Google website,opening Google Chrome just means opening the Google Chrome software.
Google Chrome can be used. The website www.google.com can not be used because Chinese Mainland cannot access it.
The user says that when searching for desired content using Google Chrome, you cannot use the URL www.google.com, but you can use Google Chrome to enter the URL of Baidu search engine, which is not contradictory.
