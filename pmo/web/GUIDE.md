## 📋 核心功能

### 1️⃣ 项目创建与 AI 解析

**操作：** 点击"新建" → 填写 → "AI 解析创建"

| 输入 | 说明 |
|------|------|
| 项目名称 | 必填 |
| 项目背景 | 目标、周期、涉及部门 |
| 负责人 | 必填 |
| 周期 (天) | 例如：60 |
| 涉及部门 | 技术部、产品部等 |

**AI 自动输出：**
- ✅ 项目摘要
- ✅ 关键信息提取
- ✅ RACI 角色分配
- ✅ 里程碑分段
- ✅ SOP 步骤
- ✅ 时间排期

---

### 2️⃣ 自动角色分配 (RACI)

**AI 按项目类型自动匹配：**

| 角色 | 说明 |
|------|------|
| R 负责人 | 对项目结果负责 |
| A 协作人 | 参与执行 |
| C 咨询人 | 提供意见 |
| I 知会人 | 接收通知 |

**支持：** 一键手动调整

---

### 3️⃣ 自动项目分段 (WBS+ 里程碑)

**AI 按行业模板拆分：**

```
阶段 1: 启动
  └─ 里程碑：需求确认

阶段 2: 规划
  └─ 里程碑：技术方案评审

阶段 3: 执行
  └─ 里程碑：开发完成

阶段 4: 收尾
  └─ 里程碑：上线发布
```

---

### 4️⃣ 自动生成 SOP

**每阶段包含：**
- 步骤
- 输入/输出
- 标准
- 验收方式

**支持：** 导出/编辑

---

### 5️⃣ 自动时间节点

**AI 按总工期倒排：**
- 开始日期
- 截止日期
- 缓冲时间
- 绑定负责人

**输出：** 甘特图关键节点

---

### 6️⃣ 自动周会总结

**输入：** 会议记录/语音转写

**输出：**
- ✅ 本周完成
- 📅 下周计划
- ⚠️ 问题
- 📝 待办 (@到人)

---

### 7️⃣ 自动提醒

**规则：** 任务截止前 3 天

**方式：**
- 📧 邮件
- 🔔 站内信

**PMO 可查看所有预警**

---

### 8️⃣ PMO 控制台

**概览：**
- 项目进度
- 逾期项目
- 预警任务
- 资源概览

---

## 🔄 工作流程

```
1. 新建项目 (输入背景)
         ↓
2. AI 解析 (目标/周期/干系人/交付物)
         ↓
3. 分配 RACI 角色 → 确认
         ↓
4. 生成里程碑 → 确认
         ↓
5. 生成 SOP → 确认
         ↓
6. 生成排期 → 确认
         ↓
    项目执行中
    ├─ 每周：自动周会总结
    └─ 到期前 3 天：自动提醒
         ↓
    项目完成/关闭
```

## 🚀 快速开始

### 第 1 步：启动服务器
```powershell
cd D:\AI-PMO-System\pmo\web
python server.py
```

### 第 2 步：打开浏览器
```
http://localhost:5000
```

### 第 3 步：创建第一个项目
1. 点击顶部"新建"
2. 填写项目信息
3. 点击"AI 解析创建"
4. 确认 AI 生成内容

### 第 4 步：查看进度
1. 点击"控制台"查看概览
2. 点击项目"查看"看详情
3. 点击"提醒"看预警

## 📱 Teams 集成

### 方式 1：SharePoint 文件
1. 复制 `dashboard.html` 到 SharePoint
2. Teams 频道 → 文件 → 打开

### 方式 2：网站标签页
1. 部署到云端获取公网 URL
2. Teams → 添加标签页 → 网站
3. 输入 URL

---

## ⚙️ 设置提醒

1. 点击侧边栏"设置"
2. 设置提前天数（默认 3 天）
3. 输入接收邮箱
4. 保存
