## 数据流

```
用户输入
   │
   ▼
┌──────────────────┐
│  命令解析        │
│  (pmo-core.ps1)  │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  OpenClaw 工具调用 │
│  - memory_search │
│  - sessions_*    │
│  - cron_*        │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  数据处理        │
│  - 读取项目文件  │
│  - 更新记忆库    │
│  - 生成报告      │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  AI 增强          │
│  - 风险识别      │
│  - 建议生成      │
│  - 模式学习      │
└────────┬─────────┘
         │
         ▼
┌──────────────────┐
│  输出            │
│  - 命令行结果    │
│  - 通知消息      │
│  - 报告文档      │
└──────────────────┘
```

## 核心模块

### 1. 项目管理模块
- 项目创建/编辑/删除
- 状态跟踪
- 里程碑管理

### 2. 风险管理模块
- 风险登记
- 影响/概率评估
- 缓解措施跟踪

### 3. 报告模块
- 周报自动生成
- 状态汇总
- 预警通知

### 4. 记忆模块
- 跨项目经验存储
- 历史决策记录
- 学习模式识别

### 5. 通知模块
- 里程碑提醒
- 风险预警
- 定期报告
