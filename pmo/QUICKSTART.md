# AI PMO 快速入门指南

欢迎使用 AI PMO 原型系统！🎉

## 🚀 5 分钟快速开始

### 1. 查看项目状态

```powershell
cd C:\Users\Administrator\.openclaw\workspace\pmo\scripts
.\pmo-core.ps1 status
```

### 2. 创建新项目

```powershell
.\pmo-core.ps1 new "你的项目名称"
```

### 3. 查看项目详情

```powershell
.\pmo-core.ps1 show PRJ-2026-001
```


## 🔧 核心功能

### 项目创建
使用模板快速创建标准化项目档案，包含：
- 基本信息（ID、负责人、状态、优先级）
- 项目目标
- 关键里程碑
- 进展跟踪
- 风险日志
- 资源需求
- 会议记录

### 状态跟踪
- 🟢 正常 - 项目按计划进行
- 🟡 风险 - 存在潜在问题
- 🔴 严重 - 需要立即关注

### 风险管理
记录、跟踪和缓解项目风险，支持：
- 风险登记
- 影响/概率评估
- 缓解措施
- 负责人分配

### 自动报告
- 周报自动生成
- 里程碑提醒
- 风险预警

---

## ⏰ 设置定时任务（可选）

编辑 `config/cron-jobs.json` 配置自动任务：

1. **每日检查** - 工作日早上 9 点
2. **周报生成** - 周五下午 5 点
3. **里程碑提醒** - 每天检查

使用 OpenClaw cron 命令激活：
```bash
openclaw cron add --job config/cron-jobs.json
```
