### 2. OneDrive 文档库配置要求

**详细文档**: `ONEDRIVE-REQUIREMENTS.md`

**核心要求**:

#### 账号要求
| 项目 | 要求 |
|------|------|
| 账号类型 | Microsoft 365 商业版/企业版 |
| 权限级别 | OneDrive/SharePoint 管理员或文档库所有者 |
| 订阅计划 | Business Basic 及以上 |

#### 必要权限
- `Sites.Read.All` - 读取站点文档
- `Files.Read.All` - 读取文件
- `User.Read` - 读取用户信息

#### 配置步骤
1. 获取 OneDrive/SharePoint URL
2. Azure AD 应用注册
3. 配置 API 权限
4. 创建客户端密钥
5. 在 Dashboard 中配置

#### 支持的文档库
- ✅ OneDrive for Business
- ✅ SharePoint Online 文档库
- ❌ 个人 OneDrive（不支持）

## ⚠️ 注意事项

### 1. 项目详情页面
- 当前使用静态数据（来自 `project-tasks-PRJ-2026-001.json`）
- 双击任务卡片可查看详细弹窗

- 实际部署应调用 OneDrive API 获取下载链接
