### 3. 核心功能实现

#### 搜索模式

**关键词搜索**
```
输入：AI 客服项目
匹配：标题、内容、元数据中包含关键词的文档
排序：按相关性得分降序
```

**自然语言提问**
```
输入：AI 客服项目的技术架构是什么？
处理：提取关键词 [AI, 客服，项目，技术架构]
识别：问题类型 = what
优化：优先返回内容丰富的文档
输出：答案摘要 + 相关文档列表
```

**高级选项**
```
- OneDrive URL 配置
- 包含子文件夹（是/否）
- 搜索文档内容（是/否）
- 仅搜索元数据（是/否）
```

#### 检索算法

```javascript
相关性得分 = 
  标题完全匹配 (10 分) +
  标题包含匹配 (5 分) +
  内容匹配 (2 分) +
  最近 7 天修改 (3 分) +
  最近 30 天修改 (1 分)
```

#### 问题类型识别

| 类型 | 关键词 | 优化策略 |
|------|--------|----------|
| what | 什么、哪些 | 优先返回内容丰富的文档 |
| how | 怎么、如何 | 优先返回方案、技术类文档 |
| why | 为什么 | 优先返回分析报告、决策文档 |
| when | 何时、什么时候 | 优先返回计划、时间表类文档 |
| who | 谁、哪个 | 优先返回人员、组织相关文档 |
