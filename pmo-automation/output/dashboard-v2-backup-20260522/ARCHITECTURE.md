## 🔄 数据流

```
用户输入
   │
   ├─→ [关键词搜索] ──→ 提取关键词 ──→ 匹配索引 ──→ 计算相关性 ──┐
   │                                                          │
   ├─→ [自然语言提问] ─→ NLP 处理 ──→ 识别问题类型 ──→ 优化结果 ─┤
   │                                                          │
   └─→ [高级选项] ────→ 应用过滤器 ────────────────────────────┘
                                                              │
                                                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    OneDrive 检索引擎                         │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐         │
│  │  文档索引   │  │  搜索算法   │  │  排序引擎   │         │
│  │  (JSON)     │  │  (关键词)   │  │  (相关性)   │         │
│  └─────────────┘  └─────────────┘  └─────────────┘         │
└─────────────────────────────────────────────────────────────┘
                                                              │
                                                              ↓
                                                  显示搜索结果
                                                      │
                                                      ↓
                                              用户点击文档
                                                      │
                                                      ↓
                                          调用 OneDrive API
                                                      │
                                                      ↓
                                                打开文档
```

## 🔌 集成点

### 当前（演示模式）
```
前端 (index.html)
   │
   └─→ JavaScript 搜索函数
         │
         └─→ 本地 JSON 索引 (document-index.json)
               │
               └─→ 模拟搜索结果
```

### 目标（生产环境）
```
前端 (index.html)
   │
   └─→ API 调用 (/api/search/*)
         │
         └─→ Node.js 后端
               │
               ├─→ Microsoft Graph API
               │     │
               │     └─→ OneDrive/SharePoint
               │
               └─→ Elasticsearch
                     │
                     └─→ 全文索引
```

## 🔐 安全架构

```
用户认证
   │
   ├─→ OAuth 2.0 (Microsoft 365)
   │     │
   │     └─→ 访问令牌
   │
   └─→ 访问控制
         │
         ├─→ 文档权限检查
         │
         └─→ 用户角色验证 (RBAC)
               │
               ├─→ 管理员：配置 + 搜索 + 管理
               ├─→ 普通用户：搜索 + 查看
               └─→ 访客：仅搜索（受限）
```

## 📈 性能优化策略

### 缓存层
```
搜索结果缓存 (Redis/Memory)
   │
   ├─→ 热门搜索查询
   │
   └─→ 常用文档内容
```

### 索引优化
```
Elasticsearch 索引
   │
   ├─→ 分词器（中文/英文）
   │
   ├─→ 同义词扩展
   │
   └─→ 拼音搜索
```

### 异步处理
```
文档扫描（后台任务）
   │
   ├─→ 消息队列 (RabbitMQ/Azure Service Bus)
   │
   └─→ 增量更新（只处理变更）
```
